StudySprint • član 2

Podatkovni model in sinhronizacija podatkov

StudySprint je progresivna spletna aplikacija za organizacijo učenja, upravljanje učnih ciljev, izvedbo study sprintov, spremljanje napredka ter skupinsko sodelovanje.

Glavne entitete in atributi

User

id, firstName, lastName, email, passwordHash, role, createdAt, lastSyncAt

Subject

id, name, description, color, userId

Goal

id, title, description, deadline, priority, status, createdAt, userId, subjectId

Task

id, title, description, deadline, status, estimatedMinutes, order, goalId

StudySprint

id, startTime, endTime, durationMinutes, status, notes, userId, taskId

Progress

id, date, completedTasks, sprintCount, totalStudyMinutes, completionPercentage, userId

Reminder

id, type, scheduledFor, status, channel, userId, goalId?, taskId?

Group

id, name, description, createdAt, createdByUserId

GroupMembership

id, userId, groupId, role, joinedAt

GroupChallenge

id, title, description, startDate, endDate, targetValue, status, groupId

Relacije

  • User 1:N Subject
  • User 1:N Goal
  • Subject 1:N Goal
  • Goal 1:N Task
  • Task 1:N StudySprint
  • User 1:N StudySprint
  • User 1:N Progress
  • User 1:N Reminder
  • User N:M Group preko GroupMembership
  • Group 1:N GroupChallenge

Sinhronizacija podatkov med odjemalcem in strežnikom

Kje je podatek Kaj vsebuje Zakaj
Vedno na napravi Aktivne naloge, lokalni vrstni red nalog, trenutno stanje sprinta, zadnji ogledani podatki. Delovanje aplikacije tudi offline in hiter odziv uporabniškega vmesnika.
Samo na strežniku Hesla (hash), sistemski audit zapisi, centralna konsolidacija skupinskih statistik. Varnost, integriteta in enoten vir resnice.
Sinhronizirano Subjects, Goals, Tasks, StudySprints, Progress, Reminders, Groups, GroupMemberships, GroupChallenges. Usklajeno stanje med več napravami in sodelovanje v skupini.

Sinhronizacija se izvaja ob prijavi, periodično (npr. na 60 sekund), ob ponovni povezavi v omrežje in ročno preko endpointa POST /api/sync.
